# adam-street-growth
Amsterdam street growth visualisation using the Adamlink API

### Demo
http://adam-streets.jelle.im/

### Install
1. run `npm install`
2. run `touch assets/js/config.js`
3. add mapbox key to config.js in this format:  ```const config = {
                                                    mapbox_key: "KEY HERE"
                                                };```
